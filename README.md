# Izi Visuals Website

A full-stack creative agency website built with React and Node.js.

## Features

### Frontend

- React + Vite
- Responsive design
- Dark/light theme
- Authentication
- Client dashboard
- Admin dashboard
- Portfolio showcase
- Services listing
- Contact form
- API integration


### Backend

- Express REST API
- MongoDB database
- JWT authentication
- User roles
- Portfolio management
- Service management
- Booking system
- Payment integration
- Email notifications
- Website settings management
